for in문

for in문은 객체의 속성(key)을 반복한다.

for in문을 배열과 사용하면 인덱스 번호가 반복된다. 하지만 인덱스 순서가 중요한 배열에 대해 in보다는 for 또는 Array.forEach 사용하기

for of문

for of문은 반복 가능한 객체의 '값을' 반복한다. 배열, 문자열, 지도, NodeList 등과 같은 반복 가능한 데이터 구조를 반복한다.
