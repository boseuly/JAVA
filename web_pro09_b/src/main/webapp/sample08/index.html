JQuery

기본 사용 방법

$(css선택자).메서드([매개변수, [매개변수 , [ ... ] ] ] ) ; (대괄호는 쓰는 거 아님)

  • $() : Jquery 를 사용하기 위해 기본적으로 사용하는 문법
  • .메서드 : Jquery에서 제공하는 다양한 메서드를 작성
  • 매개변수 : 메서드에서 요구하는 전달값, 메서드에 따라 다양한 매개변수가 사용된다.

사용 형식에서 [ ] 대괄호가 있는 경우 생략할 수 있다는 의미를 가진다.

$(CSS선택자)

  • 선택 방법 예시(3)
  • $("#ul1 > li");
  • $("#ul1");
  • $(".jq-selector")

순회(탐색) - Traversing

filtering 관련 메서드

선택자로 지정한 객체를 기준으로 객체 그룹에서 위치를 가지고 선택하는 메서드

  • $(CSS선택자). first( )
  • $(CSS선택자). eq(index)
  • $(CSS선택자). filter(CSS선택자)
  • $(CSS선택자). not(CSS선택자)
  • $(CSS선택자). last( )

ancestors 관련 메서드

선택자로 지정한 객체를 기준으로 상위 요소들을 선택할 수 있는 메서드

  • $(CSS선택자).parent()
  • $(CSS선택자).parents([CSS선택자]])
  • $(CSS선택자).parentsUntil([CSS선택자])

descendants 관련 메서드

선택자로 지정한 객체를 기준으로 하위 요소들을 선택할 수 있는 메서드

  • $(CSS선택자).children([CSS선택자])
  • $(CSS선택자).find([CSS선택자])

sideways 관련 메서드

선택자로 지정한 객체를 기준으로 형제/자매 요소들을 선택할 수 있는 메서드

  • $(CSS선택자).siblings([CSS선택자])
  • $(CSS선택자).next( )
  • $(CSS선택자).nextAll( )
  • $(CSS선택자).nextUntil(CSS선택자)
  • $(CSS선택자).prev( )
  • $(CSS선택자).prevAll( )
  • $(CSS선택자).prevUntil(CSS선택자)

is 메서드

선택자로 지정된 범위에 특정 요소가 존재하는지 찾을 때 사용하는 메서드

  • $(CSS선택자).is(CSS선택자)

extend( )

객체의 내용을 추가해주는 메소드, 겹치는 속성명(키명)은 값이 수정된다.

addClass( )

클래스명을 추가해주는 것, 덮어씌우는 게 아니라 추가가 된다.


css()

선택자에 의해 선택된 요소의 속성 값을 가져오거나 설정하는 메소드

  • $(‘요소 명‘).css(‘속성 명‘); -> 속성 명의 속성 값 리턴
  • $(‘요소 명‘).css(‘속성 명‘, ‘속성 값‘); * 속성 명의 속성 값 설정
  • $(‘요소 명‘).css(‘속성 명‘, function(index, 현재 값){}); * 함수에서 리턴되는 값으로 속성 설정
  • $(‘요소 명‘).css({오브젝트}); * 오브젝트에서 설정한 다중 속성 설정
  • css() 로 추가한 스타일을 지우고 싶을 때는removeAttr("style"); 이렇게 하면 스타일 적용한 게 전부 지워진다.
  • 인라인 css(스타일)은 되도록이면 사용하지 않기.
