Strong : 글자를 강조해주는 태그 Strong 태그는 검색엔진 또는 봇에 의해
강조된 문장을 찾을 때 즉, 강조라는 의미가 가미된 문장을 찾기 위해 사용된다. 이를 시맨틱 태그라고 한다.
b 태그 : 글자를 강조해주는 태그 b태그는 강조된 문장이라는 의미가 존재한다기 보다는
아무 의미 없이 특정 문자를 두껍게 만들어주는 것이다.
mark태그 : 글자에 형관팬으로 하이라이팅을 해주기 위해 사용한다. 태그 안에 태그를 겹쳐서 사용할 수 있다.
예) 하이라이팅과 두께를 함께 줄 수 있다.
s태그 : 취소선, u태그 : 밑줄
글자에 기울임을 적용하기 위한 em, i 태그
참고로 둘 다 똑같이 기울임을 적용하기 위해 사용하지만 em은 Strong처럼
강조하기 위한 의미로 사용할 때 사용한다.
small 태그 : 글자의 크기를 작게 표시하여 부가 정보를 표현하기 위한 태그small태그
blockquote태그 , q태그: 다른 사이트 또는 책의 문구를 인용한 경우 사용한다.
blockquote태그를 사용 하면
이렇게 한 영역을 차지
하게 된다.
반면 q태그를 사용 하면 큰따옴표 안에 문구를 넣어서 표현한다.
지금까지 살펴본 것들은 대부분 inline tag에 해당한다.
예외로 hr태그와 h1태그는 block tag에 해당한다.
